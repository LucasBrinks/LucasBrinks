# LucasBrinks - Cloudflare Pages

Este repositório está conectado ao Cloudflare Pages.

## Estrutura

```
/hello
  └── index.html  # Página principal Hello World
```

## Deploy no Cloudflare Pages

Para configurar o Cloudflare Pages:

1. Acesse o dashboard do Cloudflare Pages
2. Configure o projeto apontando para este repositório
3. Defina o diretório de build como: `hello`
4. Branch de produção: `main` ou o branch desejado

A página estará disponível após o deploy automático.
